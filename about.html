✨ ProductBuilder 소개

누구나 쉽게 즐길 수 있는 재미있고 유익한 웹 서비스를 만듭니다.

ProductBuilder는 일상에서 유용하게 활용할 수 있는 다양한 무료 웹 서비스를 제공하는 플랫폼입니다. 복잡한 회원가입 없이 누구나 즉시 이용할 수 있으며, 모바일과 PC 환경 모두에서 최적의 경험을 제공하기 위해 꾸준히 개발하고 있습니다.

🛠️ 제공 서비스

🎱
로또 번호 추천기
무작위 알고리즘으로 로또 번호를 5세트 추천해드립니다. 추천 기록도 저장하고 관리하세요.
🐶🐱
동물상 테스트
AI가 사진을 분석해 강아지상인지 고양이상인지 판별합니다. 재미있는 성격 분석도 함께!

💡 서비스 철학

  • 🆓 완전 무료 모든 서비스는 무료로 제공됩니다. 회원가입이나 결제 없이 바로 이용하세요.
  • 🔒 개인정보 보호 이미지, 번호 기록 등 민감한 정보는 서버로 전송되지 않고 기기 내에서만 처리됩니다.
  • 📱 반응형 디자인 모바일, 태블릿, PC 등 모든 기기에서 최적화된 화면으로 이용할 수 있습니다.
  • 🚀 지속적 업데이트 더 나은 서비스를 위해 지속적으로 기능을 개선하고 새로운 서비스를 추가합니다.

🤖 사용 기술

ProductBuilder의 서비스는 최신 웹 기술을 기반으로 제작되었습니다.

  • Google Teachable Machine + TensorFlow.js 동물상 테스트에 사용된 AI 이미지 분류 모델은 Google의 Teachable Machine 플랫폼을 통해 학습되었으며, TensorFlow.js를 이용해 브라우저에서 직접 실행됩니다.
  • Web Storage API (localStorage) 로또 번호 추천 기록과 테마 설정은 브라우저의 로컬 스토리지를 이용해 기기에 저장됩니다. 별도의 서버 저장소를 사용하지 않아 개인정보가 보호됩니다.
  • Disqus 댓글 시스템 사용자들이 서로 의견을 나눌 수 있도록 Disqus 댓글 시스템을 도입했습니다.
  • Formspree 제휴 문의는 Formspree 서비스를 통해 안전하게 전달됩니다.

📬 문의 및 제휴

서비스 이용 중 불편한 점이나 제휴 문의, 개선 사항 제안이 있으시면 아래 방법으로 연락해주세요. 모든 문의에 최대한 빠르게 답변드리겠습니다.

제휴 문의는 로또 번호 추천기 페이지의 제휴 문의 양식을 이용해주세요.

서비스 개선 제안이나 버그 신고도 동일한 양식으로 보내주시면 검토 후 반영하겠습니다.

📋 이용 약관 요약

ProductBuilder의 모든 서비스는 개인 이용 목적으로 무료 제공됩니다. 로또 번호 추천기는 순수 무작위 알고리즘을 사용하며 당첨을 보장하지 않습니다. 동물상 테스트 결과는 재미와 흥미를 위한 것으로 의학적·심리학적 근거가 없습니다. 서비스 이용으로 인한 어떠한 손해에 대해서도 운영자는 책임지지 않습니다. 서비스를 이용함으로써 이 약관에 동의한 것으로 간주됩니다.
